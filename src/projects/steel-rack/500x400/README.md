# steel-rack / 500x400

500x400mm スチールラック用のパーツ群。

## corner_bracket_large.scad

L字コーナーブラケット（大）。天板接続プレート付き。

- 2本の `long_bar` (147mm) をL字に組み合わせ
- 30x30x6mm の接続プレートを追加し、M4 ボルト穴 + ナットトラップで天板を固定
- 内側コーナー: R4 フィレット
- プレート外側コーナー: R8 フィレット

## corner_bracket_small.scad

L字コーナーブラケット（小）。片側が短い非対称形状。

- `long_bar` (147mm) + `short_bar` (25mm) のL字
- 短辺側は6mm分カットして壁面への干渉を回避
- 内側コーナー: R4 フィレット（3mmオフセット）

## bolt_spacer.scad

ボルトスペーサー。2つのボルト穴の間を繋ぐ小型パーツ。

- 2つの円を hull で繋いだ形状（穴中心間距離: 12.7mm）
- 片側にボルト貫通穴（6.6mm径）を配置し、穴の中心線でカット
- -Y側の底面に 45度フレアの滑り止めを追加（高さ0.5mm、幅制約 |x| <= 3.3mm）

## separator.scad

ワッシャー状のセパレーター。ボルトに通してフレーム間のスペースを確保する。

- 外径 20mm / 高さ 3.5mm / 穴径 4.6mm

## drilling_guide.scad

フレームへの穴あけ位置決めガイド。

- フレーム端にかぶせて使う段差構造（リップ: 5mm）
- 底面プレート厚 4mm + ドリルガイド穴深さ 10mm
- 角に R4 フィレット
- デフォルト: エッジから 17.5mm の位置に 4.2mm 穴
