# laundry

洗濯場で使うパーツ。

## dryer_drain_adapter.scad

乾燥機の排水ホース（オス）と洗濯機の排水口（オス）を繋ぐアダプタ。
両側がメス（穴）の筒形状で、それぞれのホースを差し込んで接続する。

### 寸法

| 側 | 接続先 | ホース外径 | 穴内径 | 収縮補正 | 挿入深さ |
|----|--------|-----------|--------|---------|---------|
| A (上) | 乾燥機 | 15.2mm | 15.8mm | +0.6mm | 35mm |
| B (下) | 洗濯機 | 14.5mm | 15.5mm | +1.0mm | 35mm |

- 壁厚: 2.5mm
- 外径は大きい方の内径に合わせて統一
- A側とB側の内径差（0.3mm）が自然なストッパーとして機能する
- A側の収縮補正は実測フィードバックにより1.0→0.6mmに調整済み
