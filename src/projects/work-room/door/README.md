# work-room / door

作業部屋のドア取り付けに使う治具・工具。

## hinge_base_alignment_jig.scad

ドア枠側のヒンジベースプレートの位置決め治具。

- ドア枠のエッジに当てて、ヒンジのネジ穴位置を決める
- 三辺（左壁・上下リップ）にガイド壁があり、枠を挟んで位置を固定
- ベース厚: 4mm

## hinge_tip_alignment_jig.scad

ドア側のヒンジ羽根（リーフ）の位置決め治具。

- base と同じ構造だが、ヒンジリーフの段差分（1.5mm）を厚みに加算
- ベース厚: 3mm + 1.5mm = 4.5mm

### 共通仕様（base / tip）

| パラメータ | 値 |
|-----------|-----|
| 治具幅 | 10mm |
| ヒンジ長 | 34mm |
| フレームリップ | 4mm |
| ガイド壁高さ | +2mm |

## drill_depth_stopper.scad

ドリルの深さ止めカラー。4mm ドリルビットに装着して穴あけ深さを制限する。

- 内径 4mm / 外径 10mm / 高さ 20mm
